Languages/Libraries: Java, Image Library, Tester Library
Date Started/Completed: May 2021

A maze game generated using an undirected connected graph with random edge weights and Kruskal's minimum spanning tree algorithm. Users are given the option to either traverse the maze themselves or allow the computer to solve it for them using either depth or breadth first search (implemented using a custom stack/queue data structure). When either the player or computer has solved the maze, the game recursively backtracks using parent nodes to display the optimal path to traverse.
